# Login Form

Login Form built using HTML, CSS and JavaScript.

![Login Form Screenshot](./Screenshot%202024-05-28%20015436.png)

## Overview

This is a simple, responsive login form implemented with plain HTML, CSS and a touch of JavaScript for form interactions/validation. It is intended as a small front-end component you can use as a starting point for learning or to drop into a static site.

## Features

- Clean, modern design using only HTML & CSS
- Basic client-side validation with JavaScript
- Easy to customize styles and markup
- Single-file implementation for quick testing (open `index.html`)

## Demo / Preview

Open `index.html` in your browser to see the login form. The screenshot above shows the default appearance.

## Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/BinaryVortex/Login-Form-14.git
   cd Login-Form-14
   ```

2. Open `index.html` in your preferred browser (double-click or use `file://` URL).

3. To modify the styles, edit the CSS inside the HTML file or extract it to a separate stylesheet.

## Files

- `index.html` — The login form and its styles/scripts.
- `Screenshot 2024-05-28 015436.png` — Preview image shown above.

## Customization tips

- Change colors by editing the CSS variables or color declarations.
- Replace placeholder logo/text with your own branding.
- Hook the form to a backend by replacing the JavaScript submit handler with an AJAX request or standard form POST.
